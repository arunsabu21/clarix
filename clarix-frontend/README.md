# Clarix Frontend

Frontend for Clarix AI Assistant built with React + Vite.

## Tech Stack

- React
- Vite
- CSS

## Setup

```
npm install
npm run dev
```

## Environment

```
VITE_API_BASE_URL= http://127.0.0.1:8000/api
```

## Notes

- Connects to Django backend API
- Clean UI System inspired by AI assistants
